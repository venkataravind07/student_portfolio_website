Project Title
Student Portfolio Website

Project Description
This project is a simple frontend website created using HTML.
It displays a student portfolio including personal information, projects, and a contact form.
This website is built using basic HTML and demonstrates simple frontend development concepts.

Target Users
Students and beginners who want to showcase their portfolio.

Page Structure
1. Home Page (index.html) – Introduction and navigation
2. About Page (about.html) – Information about the student and education
3. Projects Page (projects.html) – List of projects
4. Contact Page (contact.html) – Contact form for communication

Features
- Simple navigation menu
- Table for education details
- Ordered and unordered lists
- Image display
- Contact form
